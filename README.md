# AI Amazon Dashboard

A modern dashboard for Amazon sellers with sales metrics, charts, and AI-powered insights.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Stack

- **React 18** + **Vite**
- **Tailwind CSS**
- **Recharts** for charts
